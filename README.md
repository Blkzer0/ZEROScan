# ZEROScan

## 简介

ZEROScan 是漏洞利用框架，通过它可以很容易地获取、开发漏洞利用插件并对目标应用进行渗透测试。界面和使用借鉴了msf框架，很容易上手使用和开发插件。

## 安装

本框架采用 Python 语言开发，并且第三方依赖包都已打包，只需要下载就可以直接进行使用。
